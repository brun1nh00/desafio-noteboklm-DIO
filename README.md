# Business & Data Analytics Knowledge Base

Tema:
- Desenvolvimento de um caderno temático utilizando NotebookLM para consolidar conhecimentos essenciais das áreas de Business Analysis e Data Analytics.

Objetivos:
- Compreender o papel do Analista de Negócios.
- Compreender o papel do Analista de Dados.
- Aprender técnicas de levantamento de requisitos.
- Aprender modelagem de processos.
- Aprender indicadores de negócio (KPIs).
- Aprender análise exploratória de dados.
- Aprender visualização de dados.
- Desenvolver pensamento analítico.
- Criar uma base de conhecimento reutilizável para entrevistas e projetos.
